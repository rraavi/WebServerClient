
/*--------------------------------------------------------------READ ME-----------------------------------------------------------*/


Each Client and Server for UDP and TCP/IP and Thread Server has been compressed individually.
Test files have been included inside the Client Folders .
The make file has been included.
Extract the respective .tar.gz and run the make.
Testing has been done for TCP/IP webserver and UDP Server and the code is commented.
The programs might show an error on the first run, but then it will execute properly on the next run.
Thread Server has been implemented and tested using Browser, no seperate client has been written,if needed be, Same client as TCP/IP
is to be used.
